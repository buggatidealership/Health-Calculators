# Breast Implant Calculator — Semantic Demand Research

## Date: 2026-03-20

## People Also Ask (Google) [TRACED]
- "What size breast implant should I get?" — Most common question. Users want cc-to-cup conversion.
- "How much do breast implants cost?" — Financial planning mode. Range $5,000-$12,000 total.
- "What's the difference between silicone and saline?" — Comparison/decision mode.
- "How long do breast implants last?" — Long-term planning anxiety.
- "Do breast implants cause weight gain?" — Post-surgery concern.

## User Emotional State [TRACED from search patterns]
- **Primary emotion:** Excitement mixed with anxiety. This is an elective cosmetic procedure — users are choosing it, but the stakes feel high (permanent change to appearance).
- **Mental model:** "I want to go from X cup to Y cup" — users think in cup sizes, not cc volumes. The calculator bridges this gap.
- **Decision fatigue:** Multiple variables (type, profile, placement) overwhelm users. They want ONE answer: "what size should I get?"

## The ONE RULE [ASSUMED — synthesized from user patterns]
"Every 150-200cc equals roughly one cup size."

This is the mental anchor users need. Everything else (profile, width, cost) flows from this.

## User Language Patterns [TRACED]
- "What size implant for a C cup?" (not "what cc volume for a C cup")
- "Will I look natural?" (fear of overcorrection)
- "How much does it cost with surgeon fees?"
- "What's the right size for my body?"

## Accent Color Rationale
- Warm rose/pink (#e88da4) — supportive, feminine energy without being clinical. Matches the cosmetic/aesthetic category.

## Falsifiability
- The 150-200cc per cup claim is approximate and varies by manufacturer and body type. Evidence would disprove this if standardized sizing existed (it doesn't — [TRACED]).
- Cost estimates based on ASPS 2024 data. Would be invalid if major market shifts occurred.
