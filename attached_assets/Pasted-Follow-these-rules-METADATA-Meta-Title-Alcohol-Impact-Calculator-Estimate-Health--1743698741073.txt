    <script>
        function calculateImpact() {
            // Get input values
            const drinks = parseFloat(document.getElementById('drinks').value);
            const gender = document.getElementById('gender').value;
            const weight = parseFloat(document.getElementById('weight').value);
            const age = parseFloat(document.getElementById('age').value);

            // Scientific parameters (GBD 2023 study)
            const BASE_RISK = 0.4; // Months per drink/week
            const RISK_MULTIPLIER = 3.0; // For drinks above guidelines
            const GUIDELINES = gender === 'male' ? 14 : 7;
            const WEIGHT_FACTOR = 70;

            // Calculate impacts
            let baseImpact = drinks * BASE_RISK;
            let excessImpact = Math.max(drinks - GUIDELINES, 0) * (BASE_RISK * RISK_MULTIPLIER);
            let totalImpact = (baseImpact + excessImpact) * (WEIGHT_FACTOR/weight);
           
            // Gender adjustment
            totalImpact *= gender === 'female' ? 1.4 : 1;

            // Format results
            const years = Math.floor(totalImpact / 12);
            const months = Math.round(totalImpact % 12);
            const resultText = years > 0 ? `${years} year${years > 1 ? 's' : ''} ${months} month${months !== 1 ? 's' : ''}` : `${months} month${months !== 1 ? 's' : ''}`;

            // Display results
            let result = document.getElementById('result');
            result.style.display = 'block';
           
            // Set appropriate class based on impact level
            result.className = '';
            if (totalImpact > 12) {
                result.classList.add('danger');
            } else if (totalImpact > 6) {
                result.classList.add('warning');
            }

            result.innerHTML = `
                <h3>Results</h3>
                <p><strong>Estimated lifespan impact:</strong> ${resultText}</p>
                <p>For ${drinks} drinks/week (${gender}, ${weight}kg):</p>
                <ul>
                    <li>Base risk: ${baseImpact.toFixed(1)} months</li>
                    ${excessImpact > 0 ? `<li>Excess risk: +${excessImpact.toFixed(1)} months</li>` : ''}
                    <li>Metabolic factor: ${(WEIGHT_FACTOR/weight).toFixed(2)}x</li>
                </ul>
                <p><small>Note: Based on population-level data from global burden of disease studies. Individual results may vary.</small></p>
            `;
        }
    </script>

Include the following:

Q: What is BAC?
A: Blood Alcohol Concentration – it measures alcohol level in your bloodstream.

Q: How long does alcohol stay in your system?
A: BAC decreases by ~0.015% per hour. Most people are sober after 6–8 hours.

Q: Does alcohol affect sleep quality?
A: Yes. Even 1-2 drinks can reduce REM sleep and increase night-time awakenings.

Q: Can I exercise after drinking?
A: Mild to moderate exercise is fine if you're sober and hydrated. Avoid strenuous workouts.

Q: How do I detox alcohol faster?
A: You can't speed up liver metabolism. Hydrate well, eat, and rest.
