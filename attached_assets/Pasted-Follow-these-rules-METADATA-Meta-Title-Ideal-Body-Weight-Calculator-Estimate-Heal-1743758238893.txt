        function calculateIBW() {
            const gender = document.getElementById('gender').value;
            const unit = document.getElementById('unit').value;
            let heightInches;

            if (unit === 'imperial') {
                const feet = parseFloat(document.getElementById('feet').value) || 0;
                const inches = parseFloat(document.getElementById('inches').value) || 0;
                heightInches = feet * 12 + inches;
            } else {
                const cm = parseFloat(document.getElementById('cm').value) || 0;
                heightInches = cm * 0.3937; // Convert cm to inches
            }

            if (heightInches < 60) {
                alert('Please enter a valid height (minimum 5 feet/152 cm)');
                return;
            }

            // Devine formula calculation
            const baseWeight = gender === 'male' ? 50 : 45.5;
            const additionalWeight = (heightInches - 60) * 2.3;
            const ibw = baseWeight + additionalWeight;

            // Calculate healthy range (±5%)
            const lowerRange = ibw * 0.95;
            const upperRange = ibw * 1.05;

        }

Include the following FAQs:

Q: What is ideal body weight (IBW)?
A: IBW is a medically recommended target weight based on height and gender, often used in clinical nutrition and drug dosing.

Q: Which formula is most accurate?
A: Devine is widely used in hospitals; Robinson and Miller are alternatives. They yield slightly different results.

Q: Should I use frame size?
A: Frame size can refine estimates but is optional. Use wrist circumference to estimate frame size if needed.

Q: Is this the same as BMI?
A: No. BMI assesses weight relative to height, while IBW tries to pinpoint the healthiest baseline weight.

Q: Can I use this for children?
A: No. This tool is for adults. Use pediatric growth calculators for children.
